2. Представление информации в ЭВМ

Основные способы представления информации:

1. Числовая информация

  • Целые числа:
    • Прямой код
    • Обратный код
    • Дополнительный код
  • Вещественные числа:
    • С фиксированной точкой
    • С плавающей точкой (IEEE 754)

2. Текстовая информация

  • Кодировки символов:
    • ASCII (7 бит)
    • Unicode (UTF-8, UTF-16)
    • KOI-8 (для кириллицы)
  • Форматы: TXT, DOC, PDF

3. Графическая информация

  • Растровая графика:
    • Пиксельное представление
    • Форматы: BMP, JPEG, PNG
  • Векторная графика:
    • Математическое описание объектов
    • Форматы: SVG, AI

4. Звуковая информация

  • Аналого-цифровое преобразование:
    • Дискретизация
    • Квантование
  • Форматы: WAV, MP3, AAC

Принципы двоичного представления:

  • Вся информация в ЭВМ представляется в виде битов (0 и 1)
  • Бит - минимальная единица информации
  • Байт = 8 бит (основная адресуемая единица)
  • Слово - группа байт (обычно 2, 4 или 8 байт)

Примеры представления данных:

Данные Представление в памяти
Число 65 (целое) 01000001 (1 байт)
Буква 'A' (ASCII) 01000001 (1 байт)
Число 3.14 (float) 01000000010010001111010111000011 (4 байта IEEE 754)
Цвет RGB(255,0,0) 11111111 00000000 00000000 (3 байта)

3. Решение задач, перевод чисел

А) 1010(2) = А(10)

Развернутая форма записи:

10102 = 1×2³ + 0×2² + 1×2¹ + 0×2⁰

Вычисление:

= 1×8 + 0×4 + 1×2 + 0×1

= 8 + 0 + 2 + 0

= 1010

Ответ: 1010
