1. Основные возможности, приёмы редактирования и форматирования в MS Word. Описать основные возможности и приемы их использования.

Microsoft Word предоставляет широкий набор возможностей для редактирования и форматирования текста:

  • Редактирование текста: вставка, удаление, копирование, перемещение текста; проверка орфографии и грамматики; поиск и замена текста.
  • Форматирование символов: изменение шрифта, размера, цвета, начертания (жирный, курсив, подчеркивание), межсимвольного интервала.
  • Форматирование абзацев: выравнивание (по левому краю, по центру, по правому краю, по ширине); отступы и интервалы; маркированные и нумерованные списки.
  • Работа со стилями: применение встроенных стилей или создание собственных для быстрого форматирования документа.
  • Вставка объектов: таблицы, изображения, диаграммы, фигуры, сноски, колонтитулы, гиперссылки.
  • Работа с разделами: создание разрывов страниц и разделов, разные колонтитулы для разных разделов.
  • Рецензирование: отслеживание изменений, добавление комментариев, сравнение документов.
  • Печать и экспорт: настройка параметров страницы, предварительный просмотр, сохранение в различных форматах (PDF, HTML и др.).

Основные приемы работы: использование сочетаний клавиш для быстрого доступа к функциям, применение стилей для единообразия документа, использование шаблонов для стандартных документов.

2. Технологии Интернет. Поиск информации в сети

Основные интернет-технологии:

  • HTTP/HTTPS - протоколы передачи гипертекста
  • HTML/CSS - язык разметки и стилей для веб-страниц
  • JavaScript - язык программирования для интерактивности страниц
  • PHP, Python, Ruby - серверные языки программирования
  • SQL - язык работы с базами данных
  • FTP - протокол передачи файлов
  • SMTP/POP3/IMAP - протоколы электронной почты

Поиск информации в интернете:

  • Использование поисковых систем (Google, Яндекс, Bing и др.)
  • Применение операторов поиска:
    • Кавычки для точной фразы - "точная фраза"
    • Минус для исключения слов - кошка -собака
    • site: для поиска на конкретном сайте - site:wikipedia.org
    • filetype: для поиска файлов определенного типа - filetype:pdf
  • Использование специализированных поисковых систем (научные, товарные и др.)
  • Поиск в социальных сетях и на форумах
  • Использование каталогов и тематических подборок

Эффективный поиск требует формулировки точных запросов, проверки достоверности источников и критической оценки найденной информации.
