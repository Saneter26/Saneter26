1. Устройство компьютера. Техническая реализация компьютера

Основные компоненты компьютера:

  • Процессор (CPU) - центральное обрабатывающее устройство, выполняет вычисления и управляет работой других компонентов
  • Оперативная память (RAM) - временное хранилище данных для быстрого доступа процессора
  • Материнская плата - объединяет все компоненты компьютера, обеспечивает их взаимодействие
  • Жесткий диск (HDD/SSD) - постоянное хранилище данных
  • Видеокарта (GPU) - обработка графики и вывод изображения на монитор
  • Блок питания - обеспечивает питание всех компонентов
  • Система охлаждения - поддерживает оптимальную температуру компонентов
  • Периферийные устройства - клавиатура, мышь, монитор, принтер и др.

Техническая реализация основана на архитектуре фон Неймана, включающей: устройство ввода/вывода, арифметико-логическое устройство, устройство управления, память.

2. Виды информации. Формы представления информации. Свойства информации

Виды информации:

  • По способу восприятия: визуальная, аудиальная, тактильная, обонятельная, вкусовая
  • По форме представления: текстовая, числовая, графическая, звуковая, видео
  • По назначению: массовая, специальная, личная

Формы представления информации:

  • Аналоговая (непрерывная)
  • Цифровая (дискретная)

Свойства информации:

  • Достоверность
  • Полнота
  • Актуальность
  • Понятность
  • Ценность
  • Своевременность

3. Определить результат выполнения алгоритма при определённых значениях исходных данных при n=4, n=1

Анализ алгоритма:

  1. Алгоритм начинается с ввода значения n
  2. Проверяется условие n > 0
  3. Если условие истинно, выполняется блок "Да", иначе - блок "Нет"
  4. Результат выводится в конце

При n = 4:

  1. Вводим n = 4
  2. Проверяем условие: 4 > 0 - истина
  3. Выполняем блок "Да": y = 4 + 1 = 5
  4. Выводим результат: y = 5

При n = 1:

  1. Вводим n = 1
  2. Проверяем условие: 1 > 0 - истина
  3. Выполняем блок "Да": y = 1 + 1 = 2
  4. Выводим результат: y = 2
