Логические операции - это операции над высказываниями (логическими переменными), результатом которых также является высказывание.

2. Локальные компьютерные сети. Назначение, принципы организации. Топология сети.

Локальная компьютерная сеть (LAN - Local Area Network) - это сеть, объединяющая компьютеры и другие устройства на ограниченной территории (офис, здание, кампус).

Назначение локальных сетей:

  • Совместное использование ресурсов (принтеры, файлы, интернет-подключение)
  • Обмен данными между пользователями
  • Организация коллективной работы
  • Централизованное управление и администрирование
  • Обеспечение доступа к общим базам данных

Принципы организации локальных сетей:

  • Использование единой среды передачи данных
  • Высокая скорость передачи данных (от 10 Мбит/с до 100 Гбит/с)
  • Ограниченная географическая зона
  • Возможность использования различных протоколов
  • Относительно низкая стоимость развертывания и обслуживания

Топологии локальных сетей:

Топология Описание Преимущества Недостатки
Шина Все устройства подключены к одному кабелю Простота, дешевизна Низкая надежность, проблемы при разрыве кабеля
Звезда Все устройства подключены к центральному узлу (хабу или коммутатору) Легкость добавления устройств, высокая надежность Зависимость от центрального устройства, больший расход кабеля
Кольцо Устройства соединены в кольцо, данные передаются по кругу Равные права у всех устройств, предсказуемое время передачи Выход из строя одного устройства нарушает работу сети
Смешанная Комбинация различных топологий Гибкость, масштабируемость Сложность проектирования и администрирования

3. C помощью инструментов Excel оформите лист под названием "ИТОГИ СЕССИИ"

Для выполнения этого задания в Excel можно использовать следующие шаги:

  1. Создайте новый лист и назовите его "ИТОГИ СЕССИИ"
  2. Создайте таблицу по образцу из билета
  3. Для сортировки фамилий по алфавиту:
    • Выделите диапазон данных (без итоговых строк)
    • На вкладке "Данные" выберите "Сортировка"
    • Укажите сортировку по столбцу "Фамилии" по возрастанию
  4. Для расчета рейтинга (среднего балла):
    • В ячейке рейтинга первой строки введите формулу: =СРЗНАЧ(C2:F2)
    • Скопируйте формулу на остальные строки
  5. Для автоматического заполнения "Информации о сдаче сессии":
    • Используйте формулу: =ЕСЛИ(ИЛИ(C2=2;D2=2;E2=2;F2=2);"Не сдал";"Сдал")
    • Скопируйте формулу на остальные строки
  6. Для расчета процента сдавших каждый экзамен:
    • Используйте формулу: =СЧЁТЕСЛИ(C2:C9;">2")/СЧЁТ(C2:C9)*100 для математики
    • Аналогично для других предметов
  7. Для подсчета числа студентов, сдавших сессию:
    • =СЧЁТЕСЛИ(G2:G9;"Сдал")
  8. Для подсчета отличников:
    • =СЧЁТЕСЛИ(H2:H9;">=4.5") (если рейтинг 4.5 и выше считать отличным)
  9. Для добавления столбцов с количеством оценок:
    • Количество двоек: =СЧЁТЕСЛИ(C2:F2;2)
    • Количество троек: =СЧЁТЕСЛИ(C2:F2;3)
    • Количество четверок: =СЧЁТЕСЛИ(C2:F2;4)
    • Количество пятерок: =СЧЁТЕСЛИ(C2:F2;5)
    • Скопируйте формулы на остальные строки
