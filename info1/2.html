1. Понятие и свойства алгоритма. Основные алгоритмические конструкции. Привести примеры на каждый вид конструкции.

Алгоритм - это точная последовательность действий, направленная на решение конкретной задачи.

Свойства алгоритмов:

  • Дискретность - алгоритм состоит из отдельных шагов
  • Определённость - каждый шаг должен быть точно определён
  • Понятность - алгоритм должен быть понятен исполнителю
  • Конечность - алгоритм должен завершаться за конечное число шагов
  • Массовость - алгоритм должен быть применим к разным исходным данным
  • Результативность - алгоритм должен приводить к результату

Основные алгоритмические конструкции:

  1. Линейная (следование) - последовательное выполнение действий.

    Пример (на псевдокоде):

    начало
    ввод a, b
    сумма = a + b
    вывод сумма
конец
  2. Ветвление (условие) - выполнение разных действий в зависимости от условия.

    Пример:

    если температура > 38 тогда
    вывод "Высокая температура"
иначе
    вывод "Температура в норме"
конец если
  3. Цикл (повторение) - многократное выполнение действий.

    Пример цикла с предусловием:

    пока счетчик < 10 делать
    вывод счетчик
    счетчик = счетчик + 1
конец пока

    Пример цикла с постусловием:

    делать
    вывод счетчик
    счетчик = счетчик + 1
пока счетчик < 10

2. Виды памяти. Организация размещения и хранения информации

Виды памяти в компьютере:

  1. Оперативная память (ОЗУ - оперативное запоминающее устройство)
    • Быстрая, энергозависимая память
    • Хранит данные и команды во время работы компьютера
    • Очищается при выключении питания
  2. Постоянная память (ПЗУ - постоянное запоминающее устройство)
    • Энергонезависимая память
    • Хранит BIOS, микропрограммы
    • Только для чтения (в классическом варианте)
  3. Кэш-память
    • Очень быстрая память небольшого объема
    • Используется для ускорения доступа к часто используемым данным
  4. Внешняя память
    • Жесткие диски (HDD), твердотельные накопители (SSD)
    • Оптические диски (CD, DVD, Blu-ray)
    • Флеш-память (USB-накопители, карты памяти)
    • Магнитные ленты (для резервного копирования)

Организация размещения и хранения информации:

  • Файловая система организует хранение данных на носителях
  • Данные хранятся в виде файлов, объединенных в каталоги (папки)
  • Для ускорения доступа используются различные методы кэширования
  • В оперативной памяти данные размещаются в ячейках, каждая имеет уникальный адрес
  • Для управления памятью используются специальные алгоритмы (например, страничная организация памяти)
