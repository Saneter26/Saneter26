1. Назначение, возможности, приёмы работы в MS Excel

Назначение MS Excel: Это табличный процессор, предназначенный для работы с электронными таблицами, выполнения расчетов, анализа данных, построения диаграмм и графиков.

Основные возможности:

  • Создание и форматирование электронных таблиц
  • Выполнение математических, статистических, финансовых расчетов
  • Работа с формулами и функциями (более 400 встроенных функций)
  • Построение диаграмм и графиков различных типов
  • Сортировка и фильтрация данных
  • Сводные таблицы для анализа больших массивов данных
  • Макросы и программирование на VBA для автоматизации задач
  • Импорт и экспорт данных в различные форматы

Основные приемы работы:

  • Ввод данных в ячейки (текст, числа, даты)
  • Создание формул (начинаются со знака =, например =A1+B1)
  • Копирование формул с относительными и абсолютными ссылками
  • Использование функций (например, =СУММ(A1:A10))
  • Форматирование ячеек (шрифты, границы, заливка, числовые форматы)
  • Создание диаграмм (выделение данных → Вставка → Диаграмма)
  • Сортировка данных (Данные → Сортировка)
  • Фильтрация данных (Данные → Фильтр)

2. Сеть Интернет: структура, адресация, протоколы передачи. Способы подключения. Браузеры. Информационные ресурсы. Поиск информации. История сети Интернет

Структура Интернета: Это глобальная сеть сетей, состоящая из множества взаимосвязанных компьютерных сетей. Основные компоненты: клиенты (пользовательские устройства), серверы, маршрутизаторы, каналы связи.

Адресация:

  • IP-адреса (например, 192.168.1.1 или IPv6 2001:0db8:85a3::8a2e:0370:7334)
  • Доменные имена (например, google.com), преобразуемые в IP-адреса через DNS
  • URL (Uniform Resource Locator) - полный адрес ресурса в сети

Основные протоколы передачи:

  • TCP/IP - базовый набор протоколов Интернета
  • HTTP/HTTPS - для передачи веб-страниц
  • FTP - для передачи файлов
  • SMTP/POP3/IMAP - для электронной почты
  • DNS - для преобразования доменных имен в IP-адреса

Способы подключения:

  • Проводные: DSL, Ethernet, оптоволокно
  • Беспроводные: Wi-Fi, мобильный интернет (3G/4G/5G), спутниковый интернет

Браузеры: Программы для просмотра веб-страниц (Chrome, Firefox, Edge, Safari, Opera).

Информационные ресурсы:

  • Веб-сайты и веб-страницы
  • Файловые архивы
  • Электронная почта
  • Социальные сети
  • Облачные сервисы

Поиск информации: Осуществляется через поисковые системы (Google, Яндекс, Bing) с использованием ключевых слов, операторов поиска и фильтров.

История Интернета:

  • 1969 - ARPANET (предшественник Интернета)
  • 1983 - переход на протокол TCP/IP
  • 1989 - концепция Всемирной паутины (WWW) Тимом Бернерсом-Ли
  • 1990-е - массовое распространение Интернета
  • 2000-е - появление Web 2.0, социальных сетей
  • 2010-е - мобильный интернет, облачные технологии

3. Решение задач на представление чисел в десятичной, двоичной системе счисления

a) 10101102 → A10

Решение:

10101102 = 1×26 + 0×25 + 1×24 + 0×23 + 1×22 + 1×21 + 0×20 =

= 64 + 0 + 16 + 0 + 4 + 2 + 0 = 8610

Ответ: 8610

b) 45610 → A2

Решение (метод деления на 2):

456 ÷ 2 = 228 (остаток 0)

228 ÷ 2 = 114 (остаток 0)

114 ÷ 2 = 57 (остаток 0)

57 ÷ 2 = 28 (остаток 1)

28 ÷ 2 = 14 (остаток 0)

14 ÷ 2 = 7 (остаток 0)

7 ÷ 2 = 3 (остаток 1)

3 ÷ 2 = 1 (остаток 1)

1 ÷ 2 = 0 (остаток 1)

Записываем остатки в обратном порядке: 1110010002

Ответ: 1110010002
