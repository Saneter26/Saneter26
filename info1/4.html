1. Компьютерные вирусы, их свойства и типы. Классификация вирусов

Компьютерный вирус - это вредоносная программа, способная создавать свои копии и внедрять их в файлы, загрузочные секторы дисков и документы.

Свойства компьютерных вирусов:

  • Способность к размножению - создание своих копий
  • Скрытность - маскировка от обнаружения
  • Деструктивность - вредоносные действия
  • Способность к заражению - внедрение в другие файлы
  • Адаптивность - приспособление к защитным механизмам

Классификация вирусов:

По среде обитания:

  • Файловые (заражают исполняемые файлы)
  • Загрузочные (заражают загрузочные секторы)
  • Макровирусы (в документах Office)
  • Сетевые (распространяются по сети)

По способу заражения:

  • Резидентные (остаются в памяти)
  • Нерезидентные (не остаются в памяти)

По деструктивным возможностям:

  • Безвредные (не вредят, но занимают ресурсы)
  • Опасные (приводят к сбоям)
  • Очень опасные (уничтожают данные)

По алгоритму работы:

  • Черви (распространяются по сети)
  • Трояны (маскируются под полезные программы)
  • Руткиты (скрывают свое присутствие)
  • Шпионские (крадут информацию)

Современные типы вредоносного ПО:

  • Рекламное ПО (adware)
  • Программы-вымогатели (ransomware)
  • Кейлоггеры (фиксируют нажатия клавиш)
  • Ботнеты (сети зараженных компьютеров)

2. Алгоритмы. Свойства алгоритмов. Основные алгоритмические конструкции. Привести примеры на каждый вид конструкции.

Алгоритм - это точная последовательность действий, направленная на решение конкретной задачи.

Свойства алгоритмов:

  • Дискретность - состоит из отдельных шагов
  • Определённость - каждый шаг точно определён
  • Понятность - должен быть понятен исполнителю
  • Конечность - должен завершаться за конечное число шагов
  • Массовость - применим к разным исходным данным
  • Результативность - должен приводить к результату

Основные алгоритмические конструкции:

1. Линейная конструкция (следование)

Последовательное выполнение операций.

Пример (на псевдокоде):

начало
    ввод a, b
    сумма = a + b
    произведение = a * b
    вывод "Сумма:", сумма
    вывод "Произведение:", произведение
конец

2. Ветвление (условная конструкция)

Выбор действия в зависимости от условия.

Примеры:

Простое условие:

если x > 0 тогда
    вывод "x положительное"
конец если

Полное ветвление:

если возраст >= 18 тогда
    вывод "Доступ разрешен"
иначе
    вывод "Доступ запрещен"
конец если

Множественное ветвление:

выбор балл
    случай 5: оценка = "Отлично"
    случай 4: оценка = "Хорошо"
    случай 3: оценка = "Удовлетворительно"
    случай 2: оценка = "Неудовлетворительно"
    иначе: оценка = "Некорректный балл"
конец выбор

3. Цикл (повторение)

Многократное выполнение действий.

Примеры:

Цикл с предусловием:

пока счетчик < 10 делать
    вывод счетчик
    счетчик = счетчик + 1
конец пока

Цикл с постусловием:

делать
    вывод "Введите пароль"
    ввод пароль
пока пароль != "secret"

Цикл с параметром:

для i от 1 до 10 шаг 1
    вывод i
конец для

3. Создайте рисунок, состоящий из трех компонентов

Пример рисунка из трех компонентов (дом, дерево и солнце):

[Здесь будет изображение дома, дерева и солнца]

Описание компонентов:

  1. Дом - прямоугольник с треугольной крышей, окном и дверью
  2. Дерево - коричневый ствол с зеленой круглой кроной
  3. Солнце - желтый круг с лучами в правом верхнем углу

Для создания такого рисунка можно использовать:

  • Графические редакторы (Paint, Photoshop, GIMP)
  • Векторные редакторы (Illustrator, Inkscape)
  • Онлайн-инструменты (Canva, Sketchpad)
  • Программирование (на Python с библиотекой Turtle или Pygame)
