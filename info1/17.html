1. Периферийные устройства: клавиатура, монитор, мышь, принтер, сканер, модем, джойстик; мультимедийные компоненты. Мультимедийные приложения и оборудование

Периферийные устройства - это устройства, которые подключаются к компьютеру и расширяют его функциональные возможности:

  • Клавиатура - устройство ввода текстовой информации и команд
  • Монитор - устройство вывода визуальной информации
  • Мышь - устройство управления курсором и ввода команд
  • Принтер - устройство вывода информации на бумагу (матричные, струйные, лазерные)
  • Сканер - устройство ввода графической информации с бумажных носителей
  • Модем - устройство для передачи данных по телефонным линиям
  • Джойстик - устройство управления в компьютерных играх

Мультимедийные компоненты включают:

  • Звуковые карты
  • Колонки и наушники
  • Микрофоны
  • Веб-камеры
  • Графические акселераторы

Мультимедийные приложения - программы для работы с аудио, видео и графикой (медиаплееры, графические редакторы, программы для видеомонтажа).

Мультимедийное оборудование - проекторы, интерактивные доски, системы видеоконференцсвязи.

2. Информация. Свойства информации.

Информация - это сведения об окружающем мире и протекающих в нем процессах, воспринимаемые человеком или специальными устройствами.

Свойства информации:

  • Достоверность - соответствие информации реальному положению дел
  • Полнота - достаточность информации для принятия решений
  • Актуальность - важность информации в данный момент времени
  • Понятность - информация должна быть выражена на языке, понятном получателю
  • Ценность - полезность информации для решения конкретных задач
  • Объективность - независимость информации от чьего-либо мнения
  • Доступность - возможность получения информации нужным пользователем
  • Защищенность - невозможность несанкционированного доступа

3. Определить результат выполнения алгоритма при определённых значениях исходных данных, при n=15, n=0, n=-7

Анализ блок-схемы алгоритма:

  1. Начало алгоритма
  2. Ввод значения переменной n
  3. Проверка условия: n > 0?
  4. Если ДА:
    • Вычисление: a = n * 2
    • Вывод значения a
  5. Если НЕТ:
    • Вычисление: a = n / 2
    • Вывод значения a
  6. Конец алгоритма

Результаты выполнения алгоритма:

Входное значение n Условие n > 0 Вычисление Результат (a)
15 Да (15 > 0) a = 15 * 2 30
0 Нет (0 не > 0) a = 0 / 2 0
-7 Нет (-7 не > 0) a = -7 / 2 -3.5
