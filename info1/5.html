1. Методы защиты информации. Антивирусные программы. Защита информации в сетях

Методы защиты информации:

Организационные методы

  • Разработка политики безопасности
  • Регламентация доступа к информации
  • Обучение сотрудников
  • Физическая защита носителей

Технические методы

  • Шифрование данных
  • Антивирусная защита
  • Межсетевые экраны (firewalls)
  • Системы обнаружения вторжений (IDS)

Программные методы

  • Использование защищенных протоколов (SSL/TLS)
  • Системы контроля доступа
  • Резервное копирование
  • Цифровые подписи

Законодательные методы

  • Правовые нормы и стандарты
  • Лицензирование
  • Сертификация
  • Юридическая ответственность

Антивирусные программы:

Тип антивируса Принцип работы Примеры
Сканеры Проверка файлов по сигнатурам Dr.Web CureIt!, Kaspersky Virus Removal Tool
Мониторы Постоянная проверка в реальном времени Kaspersky Internet Security, ESET NOD32
Облачные антивирусы Анализ в облаке с использованием коллективного разума Panda Cloud Antivirus, Immunet
Поведенческие блокираторы Анализ поведения программ Windows Defender, Sophos

Защита информации в сетях:

  • VPN (Virtual Private Network) - создание защищенных туннелей
  • Firewall - фильтрация сетевого трафика
  • IDS/IPS - обнаружение и предотвращение вторжений
  • DLP-системы - защита от утечек данных
  • Аутентификация и авторизация - многофакторная аутентификация
  • Шифрование данных - SSL/TLS, IPsec, SSH

3. Самостоятельно составить алгоритм линейной конструкции и разветвляющейся конструкции. Реализовать его при конкретных, выбранных вами значениях.

Алгоритм линейной конструкции (расчет площади прямоугольника):

// Алгоритм расчета площади прямоугольника начало // Входные данные длина = 5 ширина = 3 // Вычисления площадь = длина * ширина // Вывод результата вывод "Площадь прямоугольника:", площадь конец

Реализация с выбранными значениями (5 и 3):

Площадь = 5 * 3 = 15

Алгоритм разветвляющейся конструкции (проверка числа на четность):

// Алгоритм проверки числа на четность начало // Входные данные число = 7 // Проверка условия если число % 2 == 0 тогда вывод "Число", число, "четное" иначе вывод "Число", число, "нечетное" конец если конец

Реализация с выбранным значением (7):

7 % 2 = 1, поэтому вывод: "Число 7 нечетное"

Дополнительный пример алгоритма с разветвлением (определение максимального из трех чисел):

// Алгоритм поиска максимального из трех чисел начало // Входные данные a = 12 b = 8 c = 15 // Поиск максимума если a > b и a > c тогда max = a иначе если b > c тогда max = b иначе max = c конец если // Вывод результата вывод "Максимальное число:", max конец

Реализация с выбранными значениями (12, 8, 15):

15 > 12 и 15 > 8, поэтому max = 15
