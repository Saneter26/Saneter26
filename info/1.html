1. Понятие об информации. Ее виды, свойства и роль в окружающем мире и производстве.

Информация – это сведения, сообщения, данные, знания, которые являются объектом хранения, передачи, обработки и преобразования. Информация уменьшает неопределенность наших знаний о чем-либо.

Виды информации:

  • По форме представления: текстовая, числовая, графическая, звуковая, видеоинформация.
  • По способу восприятия: зрительная, слуховая, тактильная, обонятельная, вкусовая.
  • По общественному значению: массовая, специальная, личная.
  • По области применения: научная, техническая, экономическая, социальная, бытовая и др.

Свойства информации:

  • Достоверность: информация должна соответствовать истинному положению дел.
  • Полнота: достаточность информации для принятия решения или понимания ситуации.
  • Актуальность (своевременность): информация должна быть получена и использована вовремя.
  • Объективность: независимость информации от чьего-либо мнения, суждения.
  • Ценность: полезность информации для достижения цели.
  • Понятность (доступность): информация должна быть представлена в форме, удобной для восприятия.
  • Краткость: отсутствие избыточной информации.

Роль информации в окружающем мире и производстве:
Информация является ключевым ресурсом современного общества. В окружающем мире она позволяет нам познавать законы природы, взаимодействовать с окружающей средой и адаптироваться к изменениям. В производстве информация является основой для принятия управленческих решений, оптимизации процессов, контроля качества, разработки новых продуктов и услуг. Информационные технологии значительно ускоряют эти процессы, повышая эффективность и конкурентоспособность.

2. Технология перевода из десятичной системы счисления в двоичную и наоборот.

Перевод из десятичной системы счисления в двоичную:
Для перевода целого десятичного числа в двоичную систему счисления используется метод деления на 2 с остатком. Десятичное число последовательно делится на 2, а остатки от деления записываются в обратном порядке.

Пример:Перевести число 13 (десятичное) в двоичную систему. 

                13 ÷ 2 = 6 (остаток 1)
                6 ÷ 2 = 3 (остаток 0)
                3 ÷ 2 = 1 (остаток 1)
                1 ÷ 2 = 0 (остаток 1)
                Результат: 1101₂
Для перевода дробной части десятичного числа в двоичную систему используется метод умножения на 2. Дробная часть последовательно умножается на 2, а целые части полученных произведений записываются в прямом порядке. Умножение продолжается до тех пор, пока дробная часть не станет равной нулю или до достижения необходимой точности.

Перевод из двоичной системы счисления в десятичную:
Для перевода двоичного числа в десятичную систему необходимо представить его в виде суммы произведений цифр на степени основания 2. Каждая цифра двоичного числа умножается на 2 в степени, соответствующей ее позиции (начиная с 0 для самой правой цифры целой части и -1 для первой цифры дробной части).

Пример: Перевести число 1101₂ в десятичную систему. 

                1101₂ = 1 * 2³ + 1 * 2² + 0 * 2¹ + 1 * 2⁰
                      = 1 * 8 + 1 * 4 + 0 * 2 + 1 * 1
                      = 8 + 4 + 0 + 1
                      = 13₁₀

3. Практическое задание:

Для выполнения этого задания потребуется использовать файловый менеджер (например, "Мой компьютер" / "Этот компьютер" в Windows).

  1. Создайте на диске D: папку "Студенты".
  2. Внутри папки "Студенты" создайте папку "Ваша группа" (например, "ИСП-24-1").
  3. Внутри папки "Ваша группа" создайте папку "Ваша фамилия" (например, "Иванов").
  4. Внутри папки "Ваша фамилия" создайте три папки: "База данных", "Информатика", "Презентации".
  5. Внутри папки "Информатика" создайте три папки: "Тексты", "Рисунки", "Таблицы".

Структура папок будет выглядеть следующим образом: 

                D:\
                └── Студенты\
                    └── Ваша группа\
                        └── Ваша фамилия\
                            ├── База данных\
                            ├── Информатика\
                            │   ├── Тексты\
                            │   ├── Рисунки\
                            │   └── Таблицы\
                            └── Презентации\
