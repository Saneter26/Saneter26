1. Понятие и свойства алгоритма. Основные алгоритмические конструкции.

Алгоритм — это точно определенная последовательность действий, приводящая к решению задачи за конечное число шагов.

Свойства алгоритма:

  • Дискретность (прерывность): Алгоритм состоит из отдельных, элементарных шагов.
  • Детерминированность (определенность): Каждое действие алгоритма четко определено и не допускает неоднозначного толкования. Для одних и тех же исходных данных результат всегда будет одинаковым.
  • Конечность: Алгоритм должен завершаться за конечное число шагов для любых допустимых исходных данных.
  • Массовость (универсальность): Алгоритм применим для решения целого класса однотипных задач, а не только для одной конкретной.
  • Результативность: Алгоритм должен приводить к получению определенного результата.

Основные алгоритмические конструкции:

  • Линейная конструкция (следование): Действия выполняются последовательно одно за другим.
    Пример: Вычисление суммы двух чисел. 
                            1. Ввести число А
                        2. Ввести число В
                        3. Вычислить Сумма = А + В
                        4. Вывести Сумма
  • Разветвляющаяся конструкция (условный оператор): Выполнение тех или иных действий зависит от выполнения некоторого условия.
    Пример: Определение большего из двух чисел. 
                            1. Ввести число А
                        2. Ввести число В
                        3. Если А > В, то
                           Вывести А
                           Иначе
                           Вывести В
  • Циклическая конструкция (повторение): Повторение одной и той же последовательности действий несколько раз.
    Примеры:
    • Цикл с предусловием (While): Действия выполняются, пока условие истинно. Условие проверяется перед каждым выполнением тела цикла.
      •                             1. Инициализировать счетчик i = 1
                            2. Пока i <= 5 делать:
                                 Вывести "Привет"
                                 Увеличить i на 1
    • Цикл с постусловием (Do-While/Repeat-Until): Действия выполняются хотя бы один раз, затем условие проверяется.
      •                             1. Инициализировать счетчик i = 1
                            2. Делать:
                                 Вывести "Привет"
                                 Увеличить i на 1
                                Пока i <= 5
    • Цикл с параметром (For): Действия повторяются заданное число раз.
      •                             1. Для i от 1 до 5 делать:
                                 Вывести "Привет"

2. Виды памяти. Организация размещения и хранения информации.

Компьютерная память делится на несколько видов в зависимости от скорости доступа, стоимости и назначения:

  • Оперативная память (ОЗУ, RAM - Random Access Memory): * Быстрая, энергозависимая память. * Используется для временного хранения данных и программ, которые активно используются процессором. * Содержимое ОЗУ теряется при выключении питания.
  • Постоянная память (ПЗУ, ROM - Read-Only Memory): * Энергонезависимая память, предназначенная только для чтения. * Хранит базовые программы загрузки компьютера (BIOS/UEFI) и другие системные данные. * Содержимое ПЗУ сохраняется при выключении питания.
  • Кэш-память: * Очень быстрая, небольшого объема память, расположенная между процессором и ОЗУ. * Предназначена для хранения наиболее часто используемых данных и инструкций, чтобы ускорить доступ к ним процессора. * Существуют уровни кэша: L1 (самый быстрый, встроен в ядро процессора), L2, L3.
  • [span_2](start_span)Внешние запоминающие устройства (ВЗУ) / Накопители массивов информации:[span_2](end_span) * Энергонезависимая память для долговременного хранения больших объемов данных. * Примеры:
    • Жесткие диски (HDD - Hard Disk Drive): Магнитные накопители, обеспечивающие большой объем хранения при относительно невысокой скорости.
    • Твердотельные накопители (SSD - Solid State Drive): Более быстрые и надежные, чем HDD, используют флеш-память.
    • Оптические диски (CD, DVD, Blu-ray): Используются для чтения и записи данных с помощью лазера.
    • Флеш-накопители (USB-флешки, карты памяти): Компактные, портативные устройства на основе флеш-памяти.
    • Магнитные ленты: Используются в основном для резервного копирования больших объемов данных.

Организация размещения и хранения информации: Информация в компьютере хранится в виде двоичного кода (битов и байтов). * Файловая система: Операционная система организует информацию на внешних носителях в виде файлов и каталогов (папок). Каждый файл имеет имя, расширение и атрибуты. * Логическая структура: Пользователь взаимодействует с файлами и папками через иерархическую структуру каталогов (дерево каталогов). * Физическая структура: На физическом уровне данные хранятся на дисках в виде блоков или секторов, которые адресуются контроллером диска. Операционная система управляет распределением этих блоков и отслеживает их расположение с помощью файловой системы.
